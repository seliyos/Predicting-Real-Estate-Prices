## Project Overview
This project builds an AI-based system to predict home sale prices based on property features. The goal is to help buyers, sellers and real estate professionals estimate fair market values and make data-driven decisions.

## Setup
``````bash
# Clone and navigate
git clone https://github.com/seliyos/Predicting-Real-Estate-Prices.git
cd house-price-prediction

# Install dependencies
pip install -r requirements.txt
``````

## Data
Download from: https://www.kaggle.com/competitions/house-prices-advanced-regression-techniques/data

## Usage
All experiments are implemented in a Google Colab notebook.

Steps:
Open the notebook in Google Colab  
Upload the dataset files  
Run the notebook from top to bottom  
Review evaluation metrics and visualizations  


## Explore Data
Initial data exploration was performed to understand the structure, distributions, and relationships within the dataset.

Exploration steps included:
Inspecting dataset shape and feature types  
Checking for missing values  
Analyzing the distribution of SalePrice  
Visualizing correlations between numerical features  
Identifying important features related to house prices
